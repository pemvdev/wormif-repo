# WormIf - Identificação Biológica e Metamorfose com IA

Escopo Funcional (Funcionalidades do Sistema)

    Captura e Envio de Imagem: Interface para tirar fotos ou carregar arquivos do dispositivo com foco no espécime.

    Detecção do Estágio de Vida: Classificação automatizada entre ovo, larva, ninfa, pupa ou adulto.

    Identificação da Espécie: Sugestão taxonômica baseada no estágio de vida previamente detectado.

    Resultado com Nível de Confiança: Exibição da probabilidade de acerto da IA e indicação de espécies semelhantes.

    Histórico de Análises: Consulta de imagens e diagnósticos realizados anteriormente.

    Base Validada por Especialistas: Painel para revisão e curadoria técnica feita por biólogos/entomologistas.

    Aprendizado Contínuo: Fluxo de alimentação do modelo com novas imagens validadas.

Escopo Não Funcional

Algumas funcionalidades e diretrizes técnicas tratadas como requisitos de sistema.

    Compatibilidade Mobile: Funcionamento nativo ou híbrido para Android e iOS.

    Performance de Análise: Tempo de resposta rápido para facilitar o uso em campo.

    Tratamento de Ruído Visual: Algoritmo prioriza o organismo principal, ignorando fundos complexos (folhas, galhos, solo).

    Segurança de Dados: Armazenamento protegido para as análises e base de conhecimento.

    Interface Intuitiva: Design focado em simplicidade para usuários em ambientes externos (fazendas/florestas).
